## About

Personal portfolio built as a cybersecurity-themed terminal interface. Sections cover projects, experience, skills, research publications, and a contact form — all rendered in a hacker-terminal aesthetic with animated transitions and an interactive sidebar.

---

## Featured Projects

| Project | What it does |
|---|---|
| **[AgentSentry](https://agentsentry.tool)** | Open-source CLI auditor for Non-Human Identities and AI agent attack surfaces across AWS, Azure, GCP, GitHub, and Kubernetes. Builds blast-radius attack graphs, enriches with CISA KEV data. Published on PyPI as `nhi-audit`. |
| **[CityBus Live](https://github.com/Abhiram-ops/citybus-live)** | Hardware-free real-time bus tracking deployed live on 4 Visakhapatnam routes. Repurposes drivers' smartphones via HTML5 Geolocation + WebSocket. Research paper submitted to ASIANCON 2026 & ICST 2026. |
| **Cybersweep** | Python recon toolkit — automates port scanning, service fingerprinting, and threat report generation in a single run. |
| **Web App Vulnerability Scanner** | Automates OWASP Top 10 checks (SQLi, XSS, CSRF) and generates structured remediation reports. |
| **LeadFinder** | Standalone `.exe` for automated B2B lead generation via SerpAPI + BeautifulSoup. |
| **E-Attendance** | Dual-factor attendance system (time-limited QR + OTP) that eliminates proxy attendance. |

## Tech Stack

- **Framework:** React 18 + TypeScript
- **Build:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Contact form:** Web3Forms API
- **Deployment:** GitHub Pages

---

## Run Locally

```bash
git clone https://github.com/Abhiram-ops/abhiramportfolio.git
cd abhiramportfolio
npm install
npm run dev
```

Open `http://localhost:5173`
